Features

  • Prepare athlete's card. Log number assignment
  • Read result from an athlete's card
  • Assign athlete's name to the result, that was read
  • Validate athlete's result (missing or extra control points, course time limit, etc.)
  • Import list of athletes/teams from Excel, text files, SFR Smart Terminal's or SFR Event Centre's databases
  • Import courses from OCAD text files and IOF XML v.3.0
  • Export results as HTML or CSV files (can be opened with Excel, OpenOffice or LibreOffice)
  • Program SFR base units
